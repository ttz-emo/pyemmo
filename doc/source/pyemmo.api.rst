api package
===========

This section is about the PyEMMO API(s) (Application Programming Interface).
There are actually two ways to interface external machine geometries to PyEMMO.
First there is the JSON format based API. The second one interfaces machines
created with the `Pyleecan <https://pyleecan.org/>`_ project.
