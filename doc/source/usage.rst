Usage
=====

Installation
------------

To use PyEMMO, first install it using pip (currently only .whl and .tar.gz files available for installation):

.. code-block:: console

    $ pip install pyemmo-1.2.9.tar.gz

If you have allready installed a version of pyemmo and you want to upgrade on a **new** version:

.. code-block:: console

    $ pip install --upgrade pyemmo-1.3.0.tar.gz

If you want to install a older version than the version you have currently installed use:

.. code-block:: console

    $ pip install pyemmo-1.2.6.tar.gz --force-reinstall
